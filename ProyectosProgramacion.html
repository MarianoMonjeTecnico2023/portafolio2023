Proyectos Destacados

Sistema de Gestión para Supermercado

  • Desarrollado en C# utilizando SQL Server y Entity Framework.
  • Permite el alta de usuarios con diferentes roles (administrador, cajero, etc.).
  • Gestiona el alta, baja, modificación y eliminación de productos, proveedores y clientes.
  • Genera facturas de compra y venta, incluyendo cálculos de impuestos y descuentos.
  • Implementa interfaz de usuario intuitiva y reportes personalizados.

Página Web de Reproducción de Música:

  • Consume la API REST de Spotify para obtener información de artistas, álbumes y canciones.
  • Permite reproducir fragmentos de 30 segundos de canciones disponibles.
  • Utiliza un reproductor de música moderno e integrado en la página.
